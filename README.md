# Retail Analytics Engineering Project

## Overview

This project demonstrates an end-to-end Analytics Engineering workflow using Snowflake and SQL.

The objective is to load retail sales data into a cloud data warehouse, build reporting layers, and generate business KPIs for sales and profitability analysis.

## Technologies Used

* Snowflake
* SQL
* GitHub

## Project Architecture

CSV Dataset

↓

Snowflake Table

↓

Business Views

↓

KPI Reporting

## Implemented Features

* Created Snowflake database and schema
* Loaded retail sales dataset into Snowflake
* Built analytical SQL queries
* Created business KPI views
* Performed regional sales and profitability analysis
* Implemented aggregations and reporting metrics

## Key KPIs

* Revenue by Region
* Profit by Region
* Top Selling Products
* Top Customers by Sales

## Sample Business Insights

* West region generated the highest sales revenue.
* West region generated the highest profit.
* Canon imageCLASS 2200 Advanced Copier is among the highest-selling products.
